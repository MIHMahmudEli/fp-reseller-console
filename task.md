# Task

## FlashProxy Practical Task (48 hours)

### WHAT WE DO

We're a proxy company. Resellers use our API to manage their proxy plans, balance, and usage.

Public docs: https://rapi.flashproxy.com/docs

We'll give you a live reseller API key with a real $50 balance to test with.

### YOUR TASK

Build a public reseller dashboard on top of our API.

Any reseller should be able to go to the dashboard and log in by entering their API key. Once logged in, they should be able to understand their account — their balance, their plans, and their usage and performance metrics. Make the data clear and useful, like a real product our resellers would use every day.

Add a small backend of your own between the dashboard and our API, so that we can see who is using the dashboard and what they do — including an audit log of logins and actions. How you design the auth, access control, and logging is up to you.

### HOW TO WORK

- Do it on GitHub, in a public open-source repo. Share the link when you're done.
- Commit as you go, with clear messages that explain what you did and why. We want to follow your thinking through the history, not just see the final result.
- Plan it yourself first. Decide the architecture, which endpoints matter, and what to focus on. A short note explaining your choices is welcome.
- Build a real, working slice. It doesn't need every feature — it needs to be thoughtful and show good judgment.
- Write a short README: how to run it, what you built, and what you'd do next.

### GROUND RULES

- Stack: TypeScript + Next.js. Use AI tools if you want — we care about the result and your thinking.
- The repo is public, so be careful: the API key must never be committed — keep it in environment variables / outside the code. Don't paste our API key or share private API details anywhere in the repo.
- The $100 is real money. Spend it carefully and note anything you use it on.
- Time: 48 hours.

### WHAT WE CARE ABOUT MOST

We want to see how you think, what you choose to build, and how far you push it on your own. Show us your drive.
